⚡ ACTO — AI Chief Task Officer
> Don't be reminded. Be done.

We've all been there — a deadline slips, an email goes 
unsent, a meeting gets missed. Not because we didn't 
know about it, but because actually doing something 
about it felt like too much effort in the moment.

ACTO fixes that. It's an AI agent that sits in the 
background, monitors your Gmail and Google Calendar, 
spots deadlines before they become problems, and takes 
action — drafting emails, rescheduling meetings, and 
sending responses — all with just one tap of your approval.

And if you've already missed something? ACTO doesn't 
judge. It activates Recovery Mode and builds you a 
3-step plan: apology email, extension request, and a 
catch-up invite — executed automatically so you can 
focus on what actually matters.

This isn't a to-do app. This isn't a reminder tool. 
ACTO is an executor.

 Features
- ⚡ Auto-drafts and sends emails on your behalf
- 🚨 Recovery Mode — instant 3-step fix for missed deadlines
- 🧠 Reads your Gmail threads to match tone and context
- 📅 Auto-syncs with Gmail, Google Calendar and Tasks
- 🎙️ Voice commands — speak your crisis, agent handles it
- 📊 Analytics dashboard — deadlines met, time saved, recovery rate
- 🌙 Dark and light mode with smooth transitions

Tech Stack

Frontend & UI
- React v18 — modular and performant UI
- Tailwind CSS — glassmorphism effects and responsive design
- Framer Motion — fluid animations and micro-interactions
- TypeScript — type-safe development environment

Backend & Infrastructure
- Node.js + Express — server-side logic and API routing
- Vite — fast modern build pipeline

AI & Google APIs
- Gemini 1.5 Pro — core AI agent with function calling
- Gmail API — read emails and send responses
- Google Calendar API — detect and reschedule events
- Google Tasks API — task management
- Speech-to-Text API — voice command processing
- Google OAuth 2.0 — secure authentication

Run Locally
1. Clone this repo
2. Run `npm install`
3. Add your `GEMINI_API_KEY` in `.env.local`
4. Run `npm run dev`
5. Open `http://localhost:3000`
